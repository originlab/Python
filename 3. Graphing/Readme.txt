Origin can create complex graphs. So to take full advantage of the graphing features, it is best that you prepare a project with the graphs already made and just push in new data. We have several examples of that in the Sample Projects with Attached Python Codes (https://www.originlab.com/doc/python/Sample-Projects-with-attached-Python-Code) page.
